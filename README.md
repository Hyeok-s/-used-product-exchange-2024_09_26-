# 중고물품 중개거래 웹(개인)

### 사용기술
Spring Tool Suite, java, gradle, thymeleaf, mysql, Google OAuth, spring security, Daum Post Code Service


## 개발 목적
중고 물품을 보다 안전하게 거래할 수 있는 증게거레 웹을 만들고자 하였다.<br/>
